Selektorer

Titta på videon

Skapa en enkel hemsida med html och css som handlar om dig och dina intressen.

Krav:

  • Döp filerna till index.html respektive style.css
  • Använd inlänkad css
  • Sidan ska klara att visa ÅÄÖ
  • Formatera koden som html 5
  • Infoga tre olika textlänkar till valfria sidor - Använd pseudoklassen hover för att få en mouseovereffekt på länkarna
  • Använd selektorerna tagg, id och class
  • Infoga tre olika bilder - Använd class för att ändra storlek på bilderna
  • Använd border någonstans på din sida
  • Minst en kommentar i din css-kod

Inlämning ska bestå av två filer, index.html och style.css, alternativt en zippad katalog som innehåller båda dessa.
