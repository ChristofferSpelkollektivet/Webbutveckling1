En första hemsida

För att komma igång med kursen ska vi börja med lite grundläggande html och css.

Titta på videon

Följ gärna med och gör samma sak som han gör.

Efter att du tittat klart och förstått allt så ska du göra din egna första hemsida. Om du inte har något kul att skriva om så går det bra att använda lorem ipsum-text.

Krav:

  • Döp filerna till index.html respektive style.css
  • Använd inlänkad css
  • Sidan ska klara att visa ÅÄÖ
  • Formatera koden som html 5
  • Använd taggarna h1 och p för rubrik och text
  • Infoga tre olika textlänkar till valfria sidor
  • Infoga tre olika bilder
  • Ändra bakgrundsfärg med css
  • Ändra teckenfärg med css
  • Använd en klass för att ändra utseendet på ett element i din htmlsida.

Inlämning ska bestå av två filer, index.html och style.css, alternativt en zippad katalog som innehåller båda dessa.
